# VPN Setup Guide

## VPN Error 422: Authentication Timeout

This is the most common VPN error at Oxford University.

**Immediate fix:**
1. Close Cisco AnyConnect completely
2. Wait 30 seconds
3. Reopen and try connecting again
4. Respond to MFA within 60 seconds

**If error persists:**
1. Clear AnyConnect preferences
2. Restart the application
3. Re-enter vpn.oxforduniversity.com

## VPN Installation

**Windows:**
1. Go to https://vpn.oxforduniversity.com
2. Login with domain credentials
3. Download for Windows
4. Run as Administrator
5. Install and restart

**Mac:**
1. Go to https://vpn.oxforduniversity.com
2. Download for Mac
3. Allow system extension in Security & Privacy
4. Complete installation

## MFA Setup

All VPN connections require MFA.

**Using Duo Mobile:**
1. Download Duo Mobile app
2. Scan QR code during VPN connection
3. Approve push notifications to connect

## Common Issues

**Cannot connect:** Check internet connection first
**Timeout errors:** Respond to MFA within 60 seconds
**Certificate errors:** Click Accept on first connection
