# WiFi Troubleshooting

## Cannot Connect to Corporate WiFi

**Steps:**
1. Verify connecting to AcmeCorp-Secure (not AcmeCorp-Guest)
2. Forget the network
3. Reconnect with domain credentials
4. Username: firstname.lastname@acmecorp.com
5. Password: your domain password

## Slow WiFi Speed

**Troubleshooting:**
1. Test speed at speedtest.acmecorp.com
2. Expected: 100+ Mbps download
3. Move closer to access point
4. Switch to 5GHz band if available
5. Restart device

## WiFi Drops Frequently

**Solutions:**
1. Update WiFi drivers
2. Disable power-saving for WiFi adapter
3. Avoid interference sources
4. Reset network settings

## Corporate vs Guest WiFi

**AcmeCorp-Secure:**
- Use domain credentials
- Full access to internal resources
- Faster speeds

**AcmeCorp-Guest:**
- No password required
- Internet only
- For personal devices
