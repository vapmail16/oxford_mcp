# Software Installation Policies

## Pre-Approved Software

Install from Software Center without approval:
- Microsoft Office 365
- Google Chrome
- Mozilla Firefox
- Adobe Acrobat Reader
- Zoom
- Slack
- 7-Zip

## Role-Based Software

**Engineering:**
- Visual Studio Code
- Git
- Docker Desktop
- Postman

**Design:**
- Adobe Creative Cloud
- Figma

**Finance:**
- QuickBooks
- SAP GUI

## Request Approval

For software not pre-approved:
1. Go to https://software-request.acmecorp.com
2. Fill out request form
3. Manager approval required
4. IT reviews within 3 business days

## Prohibited Software

Never install:
- BitTorrent clients
- Cryptocurrency miners
- Pirated software
- Personal cloud storage (use OneDrive)

## Installation Methods

**Preferred:** Software Center
- Automated installation
- Always latest version
- Pre-configured

**Alternative:** IT-assisted installation
- Submit request
- IT installs remotely
